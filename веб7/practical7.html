Explain “repaint” and “reflow” of the webpage.

Repaint occurs when changes are made to an element's visual styles that do not affect its layout on the page, such as color or visibility. Reflow, also known as layout, happens when there is a change in the layout due to alterations in the DOM that affect the size or position of elements, which requires the browser to re-calculate layout, causing further repaints for affected elements.

How does JavaScript on the webpage affect the page load speed and performance?

JavaScript can affect page load speed and performance as it is a blocking resource; when the browser encounters a script it must stop and execute it before it can continue parsing the HTML. Heavy JavaScript can delay the time to interactive, and inefficient code can lead to poor performance post-load due to increased memory and CPU usage.

What is “minification” of the script? Why it is useful to minify your scripts?

Minification is the process of removing unnecessary characters from the code (like whitespace, comments, etc.) without changing its functionality. It is useful because it reduces the size of the script, which decreases load times and bandwidth usage, thus improving the website's performance.

What is a JS Web Worker and when do we need it?

A JS Web Worker is a JavaScript script that runs in the background, in its own thread, without affecting the performance of the page. We need it for performing heavy tasks without interrupting the user interface, thereby improving the user experience on complex web applications.
