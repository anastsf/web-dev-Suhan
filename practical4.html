Questions

What is selector’s specificity and who it’s calculated?

Selector specificity determines which CSS rule takes precedence when multiple rules apply to the same element based on the hierarchy of selector types: inline > ID > class/pseudo-class > type > universal. Complex selectors combine multiple simple selectors to target elements more specifically, but can increase specificity and affect rule priority. Understanding specificity helps avoid styling conflicts.

What are complex selectors?

Complex selectors in CSS are combinations of multiple simple selectors that are used to target elements with more precision. They allow you to specify exactly which elements should be styled based on various criteria like their parent-child relationships, attributes, or states. This can help you apply styles more selectively to certain elements on your page.

What is the difference between pseudo-classes and pseudo-elements?

Pseudo-classes target special states or interactions of elements (e.g., :hover, :active), while pseudo-elements style specific parts of elements or create virtual elements (::before, ::after).

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.
